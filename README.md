### Installation
1. Clone the repo :
    ``` sh
    git clone git@github.com:casl0x/todolist.git
    ```
2. Install NPM packages :
    ```sh
    npm install
    ```
3. Start your local development server :
    ```sh
   npm run dev 
   ```
4. Have fun ;)
